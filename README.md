## Project info

FYNXX:
A social-commerce platform that connects micro-influencers with brands through a social currency model. Influencers browse partner brands, purchase using a social currency card, post content, and earn cashback based on their social score.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
